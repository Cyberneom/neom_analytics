# Changelog

All notable changes to neom_analytics will be documented in this file.

## [1.3.0] - 2025-02-09

### Changed
- Replaced deprecated `withOpacity()` with `withValues(alpha:)` in chart components
- Updated yearly_line_chart.dart gradient colors
- Updated weekly_bar_chart.dart background color

### Improved
- README.md with comprehensive documentation and ROADMAP 2026
- Code compliance with latest Flutter/Dart deprecation guidelines

## [1.2.0] - 2025-01-15

### Added
- YearlyLineChart with 12-month visualization
- WeeklyBarChart with 7-day breakdown
- Smart Y-axis formatting (K/M suffixes)
- Average mode toggle for line charts

### Changed
- Updated flutter_lints to ^6.0.0
- SDK constraint to >=3.8.0 <4.0.0

## [1.1.0] - 2024-11-01

### Added
- fl_chart integration for flexible charting
- flutter_animate for smooth transitions
- pie_chart widget support
- CSV export capabilities

## [1.0.0] - 2024-08-15

### Added
- Initial release
- Basic chart components
- Integration with neom_core and neom_commons
